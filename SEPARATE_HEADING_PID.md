# 独立航向控制系统 / Separate Heading Control System

## 🎯 架构改进 / Architecture Improvement

### 原来的问题 / Previous Problem

```
用户命令 → 设置目标速度 → MPU 校正直接修改目标速度 → 电机 PID
                                    ↑
                                干扰 PID 积分项
```

**问题**: MPU 校正每 10ms 修改目标速度，导致电机 PID 的积分项无法稳定。

### 新的架构 / New Architecture

```
用户命令 → 基础速度
              ↓
         航向 PID ← MPU6050 pitch
              ↓
         差速输出
              ↓
    基础速度 ± 差速 → 电机目标速度 → 电机 PID → PWM 输出
```

**优势**:
1. ✅ **两个独立的 PID 控制器**
   - 航向 PID: 控制左右差速，保持直线
   - 电机 PID: 控制电机速度跟踪

2. ✅ **电机 PID 不受干扰**
   - 目标速度平滑变化（基础速度 + 缓慢变化的差速）
   - 积分项可以正常工作

3. ✅ **航向控制更精确**
   - 专门的 PID 参数调节航向
   - 有积分项消除稳态误差
   - 有微分项抑制震荡

## 📊 航向 PID 参数 / Heading PID Parameters

### 默认参数

```c
Kp = 0.5    // 比例系数
Ki = 0.05   // 积分系数
Kd = 0.1    // 微分系数
输出限幅 = ±20%  // 最大差速
```

### 参数含义

| 参数 | 作用 | 调整建议 |
|------|------|---------|
| **Kp** | 响应速度 | 太大→震荡；太小→响应慢 |
| **Ki** | 消除稳态误差 | 太大→超调；太小→偏差残留 |
| **Kd** | 抑制震荡 | 太大→噪声敏感；太小→震荡 |
| **输出限幅** | 最大差速 | 限制左右轮速度差 |

### 调整指南

#### 如果车辆左右摇摆（震荡）
```bash
# 降低 Kp 和 Kd
PID 0.3 0.05 0.05
```

#### 如果车辆偏离后不回正
```bash
# 增加 Ki
PID 0.5 0.1 0.1
```

#### 如果车辆响应太慢
```bash
# 增加 Kp
PID 0.8 0.05 0.1
```

#### 如果车辆对噪声敏感
```bash
# 降低 Kd
PID 0.5 0.05 0.02
```

## 🔧 工作原理 / How It Works

### 控制循环（每 10ms）

```c
1. 读取 MPU6050 pitch 角度
   ↓
2. 航向 PID 计算
   error = target_pitch - current_pitch
   integral += error * dt
   derivative = (error - last_error) / dt
   output = Kp*error + Ki*integral + Kd*derivative
   ↓
3. 应用差速
   右侧电机 = base_speed + output
   左侧电机 = base_speed - output
   ↓
4. 电机 PID 跟踪目标速度
   ↓
5. PWM 输出
```

### 示例计算

假设：
- 基础速度 = 50%
- Pitch 角度 = 5° (车头左偏)
- 航向 PID 输出 = -2.5% (负值表示需要右转)

结果：
```
右侧电机目标 = 50% + (-2.5%) = 47.5%
左侧电机目标 = 50% - (-2.5%) = 52.5%
```

左侧轮速度更快，车辆右转，修正偏离。

## 🎓 PID 调参技巧 / PID Tuning Tips

### Ziegler-Nichols 方法

1. **设置 Ki=0, Kd=0**
   ```bash
   PID 0.5 0 0
   ```

2. **逐步增加 Kp 直到震荡**
   ```bash
   PID 1.0 0 0
   PID 1.5 0 0
   PID 2.0 0 0  # 假设这里开始震荡
   ```

3. **记录临界 Kp (Ku) 和震荡周期 (Tu)**
   - Ku = 2.0
   - Tu = 测量震荡周期（秒）

4. **计算 PID 参数**
   ```
   Kp = 0.6 * Ku = 0.6 * 2.0 = 1.2
   Ki = 2 * Kp / Tu
   Kd = Kp * Tu / 8
   ```

### 经验调参法

1. **从小参数开始**
   ```bash
   PID 0.1 0 0
   ```

2. **增加 Kp 直到响应快速但不震荡**
   ```bash
   PID 0.3 0 0
   PID 0.5 0 0
   ```

3. **添加 Ki 消除稳态误差**
   ```bash
   PID 0.5 0.05 0
   PID 0.5 0.1 0
   ```

4. **添加 Kd 抑制震荡**
   ```bash
   PID 0.5 0.1 0.05
   PID 0.5 0.1 0.1
   ```

## 🐛 故障排查 / Troubleshooting

### 问题 1: 仍然抖动

**可能原因**:
1. 电机 PID 参数不合适（在 main.c 中）
2. 硬件问题（编码器、电源）
3. 航向 PID 参数太激进

**解决**:
```bash
# 先禁用航向校正测试
MPU OFF
RUN 30

# 如果仍抖动，是电机 PID 问题
# 需要修改 main.c 中的电机 PID 参数
```

### 问题 2: 车辆左右摇摆

**原因**: 航向 PID 震荡

**解决**:
```bash
# 降低 Kp 和 Kd
PID 0.2 0.05 0.02
```

### 问题 3: 车辆偏离不回正

**原因**: 积分项太小

**解决**:
```bash
# 增加 Ki
PID 0.5 0.15 0.1
```

### 问题 4: 车辆过度校正

**原因**: Kp 太大

**解决**:
```bash
# 降低 Kp
PID 0.3 0.05 0.1
```
